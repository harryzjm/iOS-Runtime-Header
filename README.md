# iOS-Runtime-Header

### Require  

class-dump: [3.5](https://github.com/nygard/class-dump)

### Support  

Xcode: 11

##  Usage

```bash  
chmod +x class-dump
chmod +x generate_headers.sh
./generate_headers.sh
```
